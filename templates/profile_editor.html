Profil développeur

Le profil trop flexible

Alice peut modifier son profil via une API REST utilisée par le front. En revanche, son rôle devrait rester contrôlé uniquement par le serveur et par un vrai workflow d'administration.

Swagger interne

Le front utilise aussi une API REST pour sauvegarder le profil. La documentation interne indique seulement deux champs modifiables.

PATCH /api/profile
Content-Type: application/json

{
  "email": "{{ user.email }}",
  "team": "{{ user.team }}"
}

Console API

Modifie le JSON puis envoie la requête à l'API. Le serveur devrait ignorer tout champ qui ne fait pas partie de la documentation.
